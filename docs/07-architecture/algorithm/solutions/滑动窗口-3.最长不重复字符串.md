# 滑动窗口-3.最长不重复字符串

滑动窗口-3.最长不重复字符串

class Solution {
  public int lengthOfLongestSubstring(String s) {
      int length = s.length();
      int maxLength = 0;
      // 记录不重复的字符
      Set<Character> uniqueSet = new HashSet<>();

      int rightIndex = -1;
      for (int i = 0; i < length; i++) {
          // 左指针右移则剔除最左侧元素，直到不重复能进入while循环位置，否则一直减
          if (i != 0) {
              uniqueSet.remove(s.charAt(i - 1));
          }

          while (rightIndex + 1 < length && !uniqueSet.contains(s.charAt(rightIndex + 1))) {
              uniqueSet.add(s.charAt(rightIndex + 1));
              rightIndex++;
          }
          maxLength = Math.max(maxLength, rightIndex + 1 - i);
      }

      return maxLength;
  }
}

助记：双指针+滑动窗口，前者仅操作减，后者操作加，无法加则移动前者指针进行减，重新尝试加。

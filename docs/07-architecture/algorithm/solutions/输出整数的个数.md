# 输出整数的个数

输出整数的个数
输入一个整数，输出该数二进制表示中1 的个数。其中负数用补码表示。
解题思路
负数是补码表示
>>> 为无符号右移，>>为有符号右移，当 n 为负数是会增加多余的1

ackage com.study.leet;

public class BinaryPrint {
    
    public int NumberOf1(int n) {

        int mask = 0x01;

        int count = 0;
        int t = n;
        while (t != 0) {
            if ((t & mask) != 0) {
                count++;
            }
            t >>= 1;
        }
        return count;
    }

    public static void main(String[] args) {
        BinaryPrint binaryPrint = new BinaryPrint();
        int n = 10;
        int count = binaryPrint.NumberOf1(n);
        System.out.println("二进制表示中1的个数：" + count);
    }

}

# 旋转数组的最小数字

旋转数组的最小数字
题目：
把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。 输入一个非减排序的数组的一个旋转，输出旋转数组的最小元素。 例如数组{3,4,5,1,2}
为{1,2,3,4,5}的一个旋转，该数组的最小值为1。 NOTE：给出的所有元素都大于0，若数组大小为0，请返回0。

package com.study.leet;

/**
 * 旋转数组的最小元素问题
 * 理解：
 * 1. 非减排序数组：指数组中的元素是按照不递减的顺序排列的，即每个元素都大于等于前一个元素
 *    （注意：非减排序允许有重复元素，例如{1,2,2,3,3}也是一个非减排序数组）
 * 2. 数组的旋转：把一个数组最开始的若干个元素搬到数组的末尾
 *    例如，将{1,2,3,4,5}的前3个元素搬到末尾，得到旋转数组{3,4,5,1,2}
 * 3. 问题要求：给定一个非减排序数组的旋转，找出旋转数组中的最小元素
 */
public class RotateArrayMin {
    /**
     * 寻找旋转数组中的最小元素
     * @param array 旋转后的非减排序数组
     * @return 数组中的最小元素
     */
    public int minNumberInRotateArray(int[] array) {
        // 处理边界情况
        if (array == null || array.length == 0) {
            return 0;
        }
        
        // 如果数组只有一个元素，直接返回该元素
        if (array.length == 1) {
            return array[0];
        }
        
        // 如果数组没有旋转（仍是非减排序），则第一个元素就是最小的
        if (array[0] < array[array.length - 1]) {
            return array[0];
        }
        
        // 使用二分查找来寻找最小元素
        int left = 0;
        int right = array.length - 1;
        
        // 二分查找循环
        while (left < right) {
            // 计算中间位置
            int mid = left + (right - left) / 2;
            
            // 情况1：中间元素大于右侧元素，说明已经不是递增了，属于被移动的之一，最小值在右侧
            if (array[mid] > array[right]) {
                left = mid + 1;
            }
            // 情况2：中间元素小于右侧元素，说明也许当前还在递增，最小的值还可能在左侧
            else if (array[mid] < array[right]) {
                right = mid;
            }
            // 情况3：中间元素等于右侧元素，无法确定最小元素在哪一侧，缩小右边界
            else {
                right--;
            }
        }
        
        // 当left==right时，找到最小元素
        return array[left];
    }
    
    // 测试方法
    public static void main(String[] args) {
        RotateArrayMin solution = new RotateArrayMin();
        
        // 测试用例1：正常旋转数组
        int[] array1 = {3, 4, 5, 1, 2};
        System.out.println("测试用例1结果: " + solution.minNumberInRotateArray(array1)); // 预期输出: 1
        
        // 测试用例2：没有旋转的数组
        int[] array2 = {1, 2, 3, 4, 5};
        System.out.println("测试用例2结果: " + solution.minNumberInRotateArray(array2)); // 预期输出: 1
        
        // 测试用例3：有重复元素的旋转数组
        int[] array3 = {2, 2, 2, 0, 1};
        System.out.println("测试用例3结果: " + solution.minNumberInRotateArray(array3)); // 预期输出: 0
        
        // 测试用例4：边界情况
        int[] array4 = {1};
        System.out.println("测试用例4结果: " + solution.minNumberInRotateArray(array4)); // 预期输出: 1
        
        // 测试用例5：空数组
        int[] array5 = {};
        System.out.println("测试用例5结果: " + solution.minNumberInRotateArray(array5)); // 预期输出: 0
    }
}

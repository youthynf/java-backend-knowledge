# Java 后端面试知识库

面向 Java 后端工程师的系统化复习手册：基础、框架、数据库、中间件、分布式、微服务、架构设计、网络、算法与 DevOps。每篇文章按「核心概念 → 标准回答 → 实现原理 → 代码示例 → 实战场景 → 深挖追问 → 易错点 → 总结 → 参考资料」组织，兼顾面试表达与生产落地。

## 阅读建议

- **面试冲刺**：先看「标准回答」和「深挖追问」两节，对照高频考点查漏补缺。
- **生产落地**：重点看「实战场景」和「易错点」，结合日志、监控、压测验证方案。
- **深度理解**：通读「核心概念」和「实现原理」，建立完整的知识链路。

如何解决哈希冲突问题？
解决哈希冲突方法：
链地址法：通过链表或者其他数据结构来存储发生冲突的键值对，并将他们链在同一个哈希桶上；
开放寻址法：在哈希表中找到另一个可用的位置，而不是存储在链表中，常见的有线性探测、二次探测、双重散列。
重哈希：发生哈希冲突时，通过调用另一个哈希算法重新计算新的哈希值，直到找到一个可用的哈希桶为止；
哈希桶扩容：当哈希冲突过多时，采用动态扩容的方式，增加哈希桶的数量，重新分配键值对，降低哈希冲突概率；

开放寻址法：
线性试探：当发生哈希冲突时，从冲突的为止开始，一次检查下一个位置，直到找到一个可用的位置位置位置，如果已经到达哈希表的末尾，则从表头继续试探，如果还是找不到，则进行扩容；
二次试探：解决线性试探的哈希key聚集问题，试探的步长通过二次函数计算；
双重散列：进一步解决哈希key聚集问题，试探步长采用哈希函数进行计算，随机性更好；

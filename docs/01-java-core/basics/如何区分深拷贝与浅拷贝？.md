# 如何区分深拷贝与浅拷贝？

如何区分深拷贝与浅拷贝？
1. 浅拷贝
定义：创建一个新的对象，然后将当前对象的非静态字段复制到新对象。
特点：如果字段是值类型，执行逐位复制；如果字段是引用类型，则复制引用，但不复制引用的对象，原始对象和副本对象引用相同的子对象；
实现：实现Cloneable接口并重写clone()方法。

2. 深拷贝
定义：创建一个新对象，并递归复制当前对象及所有子对象。
特点：完全独立的副本，原始对象和副本对象不共享任何子对象，修改副本对象不会影响原始对象。
实现：手动递归克隆所有引用对象，或者通过对象序列化和反序列化实现。
助记：原始对象与副本对象的引用子对象是否一致，决定是浅拷贝还是深拷贝。

# 设计模式

本目录从 Java 实现视角讲解 GoF 23 种设计模式，每篇包含 UML 结构、可运行代码、JDK/Spring 应用案例、相似模式对比和易错点。与 [07-architecture/patterns/](../../07-architecture/patterns/) 偏架构视角互补，本目录偏 Java 落地。

## 目录

### 概述

- [设计模式有哪些](设计模式有哪些？.md) — GoF 23 种模式分类、选型思路、易混淆对子

### 创建型模式（5 种）

- [单例模式如何实现](单例模式如何实现？.md) — 五种写法对比与防反射/防序列化破坏
- [原型模式怎么克隆对象](原型模式怎么克隆对象？.md) — 浅拷贝与深拷贝、`Cloneable` 的代价
- [工厂方法模式怎么用](工厂方法模式怎么用？.md) — 子类决定实例化哪个产品，Spring `FactoryBean`
- [抽象工厂模式怎么用](抽象工厂模式怎么用？.md) — 创建一族相关产品，JDBC `Connection` 是经典
- [建造者模式如何组装对象](建造者模式如何组装对象？.md) — 分步构造复杂对象，Lombok `@Builder`

### 结构型模式（7 种）

- [适配器模式怎么转换接口](适配器模式怎么转换接口？.md) — 接口转换，`InputStreamReader` 是典型
- [桥接模式如何分离维度](桥接模式如何分离维度？.md) — 多维度变化解耦，JDBC `DriverManager` 桥接
- [组合模式如何表示树形结构](组合模式如何表示树形结构？.md) — 统一处理叶子和容器，AWT `Container`
- [装饰器模式如何动态增强](装饰器模式如何动态增强？.md) — 层层包装加功能，`BufferedInputStream`
- [外观模式如何简化子系统](外观模式如何简化子系统？.md) — 子系统统一入口，`JdbcTemplate`/SLF4J
- [享元模式如何共享对象](享元模式如何共享对象？.md) — 共享内部状态，`Integer.valueOf`/String 池
- [代理模式如何控制访问](代理模式如何控制访问？.md) — 控制访问，Spring AOP/`@Transactional`

### 行为型模式（11 种）

- [责任链模式如何传递请求](责任链模式如何传递请求？.md) — 请求沿链传递，Servlet Filter/Spring 拦截器
- [命令模式如何封装请求](命令模式如何封装请求？.md) — 请求对象化支持撤销，`Runnable`/`TransactionCallback`
- [解释器模式如何解析语法](解释器模式如何解析语法？.md) — 解释语言句子，SpEL/正则/规则引擎
- [迭代器模式如何遍历聚合](迭代器模式如何遍历聚合？.md) — 遍历不暴露内部，`Collection.iterator()`
- [中介者模式如何解耦同事](中介者模式如何解耦同事？.md) — 网状耦合变星形，`DispatcherServlet`
- [备忘录模式如何保存状态](备忘录模式如何保存状态？.md) — 状态快照可恢复，事务 `savepoint`
- [观察者模式如何广播变更](观察者模式如何广播变更？.md) — 一对多通知，Spring `ApplicationEvent`
- [状态模式如何切换行为](状态模式如何切换行为？.md) — 状态决定行为，订单状态机
- [策略模式怎么替换算法](策略模式怎么替换算法？.md) — 算法可互换，`Comparator`/Spring Map 注入
- [模板方法模式如何定义骨架](模板方法模式如何定义骨架？.md) — 父类定骨架子类填空，`AbstractList`/`HttpServlet`
- [访问者模式如何分离操作](访问者模式如何分离操作？.md) — 元素稳定操作多变，ASM `ClassVisitor`

## 学习路径

1. 先读 [设计模式有哪些](设计模式有哪些？.md) 建立整体认知。
2. 按创建型 → 结构型 → 行为型顺序学习，每篇对照 JDK/Spring 案例理解。
3. 重点掌握易混淆对子：工厂方法 vs 抽象工厂、适配器 vs 装饰器 vs 代理、状态 vs 策略、外观 vs 中介者。
4. 配合 [07-architecture/patterns/](../../07-architecture/patterns/) 的架构视角，理解模式在框架中的落地。

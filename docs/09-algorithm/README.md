# 算法

算法模块用于准备 Java 后端常见笔试和面试题。后端面试不一定追求竞赛难度，但要求能稳定写出常见数据结构和中等难度算法。

## 复习重点

- **数组与双指针**：两数之和、三数之和、滑动窗口。
- **链表**：反转链表、合并链表、环形链表、删除倒数第 N 个节点。
- **栈与队列**：括号匹配、单调栈、最小栈、队列实现栈。
- **哈希表**：去重、频次统计、快速查找。
- **二叉树**：前中后序遍历、层序遍历、最近公共祖先、路径和。
- **排序与二分**：快速排序思想、归并排序、二分边界。
- **动态规划**：背包、打家劫舍、最长递增子序列、最长公共子序列。

## 面试官想考什么

- 是否能把问题抽象成合适的数据结构；
- 是否能分析时间复杂度和空间复杂度；
- 是否能处理边界条件；
- 是否能写出清晰、可维护的代码。

## 标准回答模板

1. 复述题意，确认输入输出和边界；
2. 给出暴力解法，说明复杂度；
3. 优化思路，例如哈希、双指针、二分、动态规划；
4. 写代码时注意空数组、重复元素、越界和整数溢出；
5. 最后用样例走一遍。

## 易错点

- 只背题解，不会解释为什么这样做；
- 二分边界写错；
- 忽略空指针、空数组、单元素数组；
- 动态规划状态定义不清，转移方程靠猜。

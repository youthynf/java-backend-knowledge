# 用户空间的内存布局是什么？

用户空间的内存布局是什么？
用户空间内存，地址从高到低依次是以下7种不同的内存段：
栈段：包括局部变量、函数调用上下文等。栈的大小是固定的，一般是8MB，系统支持参数配置；
文件映射段：包括动态库、共享内存等；
堆段：包括动态分配的内存，从低地址开始向上增长；
BSS段：包括未初始化的静态变量和全局变量；
数据段：包含已初始化的静态常量和全局变量；
代码段：包含二进制可执行代码；
保留区：不可访问，主要用于防止程序因为出现bug，导致读写了一些小内存地址的数据；

在这7个内存段中，堆和文件映射段的内存时动态分配的，比如c的标准库malloc() 或者 mmap()就可以分别在堆和文件映射段动态分配内存。

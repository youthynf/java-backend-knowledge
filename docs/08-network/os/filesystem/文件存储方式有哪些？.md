# 文件存储方式有哪些？

文件存储方式有哪些？
文件的数据是要存储在硬盘上面的，数据在磁盘上的存放方式，就像程序在内存中存放的方式那样，有以下两种：
连续空间存放方式
文件存放在磁盘连续的物理空间中，文件数据紧密相连，读写效率高。前提是知道文件的大小，文件头里需要指定起始块的位置和长度。存在磁盘空间碎片和文件长度不易扩展的缺陷。

非连续空间存放方式
分为链式方式和索引方式。
链表方式：存放的是离散的，不用连续的，于是可以消除磁盘碎片，同时文件的长度可以动态扩展。链表方式进一步可以分为隐式链表和显式链表两种形式，隐式是指在文件头中保存起始数据块和末尾数据块的指针地址，并且每个数据块里面留出一个指针空间来存放下一个数据块的位置，从而实现顺序遍历数据块。显式链接是使用存放在内存的链接表（文件分配表FAT），保存磁盘上所有块的链接关系，该表在整个磁盘仅设置一张，每个表项中存放链接指针，指向下一个数据块号。
索引方式：为每个文件创建一个「索引数据块」，里面存放的是指向文件数据块的指针列表，即多个文件的指针，指向具体多个数据块。文件头需要包含指向「索引数据块」的指针，这样就可以通过文件头知道索引数据块的位置，再通过索引数据块里的索引信息找到对应的数据块。
如果文件很大，大到一个索引数据块放不下索引信息，这时又要如何处理大文件的存放呢？
「链式索引块」：在索引数据块留出一个存放下一个索引数据块的指针，相当于一个索引块存不下就用多个存，同时通过隐式链的方式将这些索引块串联起来；
「多级索引块」：通过一个索引亏来存放多个索引数据块。

## 复习建议

- **能讲清攻击链路**：从攻击入口、利用条件到影响范围完整描述。
- **能区分相似攻击**：XSS vs CSRF、DNS 劫持 vs HTTP 劫持、重放 vs 中间人。
- **能落地防御**：每个攻击至少给出 2-3 种工程防护手段及取舍。
- **能补全相邻知识**：HTTPS/TLS 握手、JWT、Cookie/Session 在 [应用层](../application-layer/) 目录。

# IPV4和IPV6有什么区别？

IPV4和IPV6有什么区别？
IPV4特点：
32位正整数，分为四组，每组8个位，组与组之间通过“.”进行分隔；
首部组成：版本、首部长度、服务区分、总长度、标识、标记、片偏移、生存时间TTL、协议类型、首部校验和、源地址、目的地址、可选字段、填充；

IPV6特点：
64位正整数，同样分为四组，每组16位，组与组之间通过“:”进行分隔；
首部组成：版本、通信、流标号、有效数据长度、下一个首部、跳数限制、源地址、目的地址；
自动配置，不需要DHCP也能自动分配IP地址；
包头包头部字段固定40字节长度，取消了首部校验和，简化了首部，提高了传输效率；
有应对伪造网络IP的功能和防窃听功能，提高了安全性能。

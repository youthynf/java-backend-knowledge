# 如何提升TCP性能？

如何提升TCP性能？
提升TCP性能需从协议参数优化、网络架构适配、应用层配合三个核心维度入手，关键手段如下：
优化TCP核心参数：增大TCP窗口（启用TCP Window Scaling，突破64KB限制），提升吞吐量；更换拥塞控制算法（如BBR、CUBIC，替代传统Reno，更适配高带宽延迟网络）。
减少网络传输损耗：匹配MTU与MSS（确保MSS = MTU - TCP/IP头部开销，避免IP分片）；复用TCP连接（如HTTP/2多路复用，避免频繁建连/断连的3次握手/4次挥手开销）。
优化应用层行为：减少小包发送（合理使用Nagle算法，避免“小包风暴”，但需注意与延迟敏感场景的平衡）；对传输数据做压缩（如gzip，从源头减少传输量）。

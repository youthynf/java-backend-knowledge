# Spring

Spring 是 Java 后端面试中最核心的框架主题之一。复习时建议围绕 **容器、代理、事务、扩展点** 四条主线展开。

## 当前文章

- [Spring IOC 与 AOP 核心原理](ioc-aop.md)

## 高频问题

- 什么是 IOC？它解决了什么问题？
- Bean 的生命周期有哪些关键阶段？
- Spring 如何解决循环依赖？有哪些解决不了？
- AOP 的底层原理是什么？JDK 动态代理和 CGLIB 有什么区别？
- 为什么 `@Transactional` 有时会失效？

## 复习建议

回答 Spring 问题时，不要停留在“用了某个注解”，而要说明：

1. 注解如何被 Spring 扫描和解析；
2. 容器如何创建、注入、初始化 Bean；
3. 代理对象如何拦截方法调用；
4. 项目中哪些场景依赖这些机制。

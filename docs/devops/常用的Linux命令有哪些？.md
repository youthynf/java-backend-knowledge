# 常用的 Linux 命令有哪些？

## 核心概念

Java 后端常用 Linux 命令主要服务于部署和排障：文件查看、日志检索、进程管理、端口排查、资源监控、磁盘分析和网络诊断。

## 面试官想考什么

- 是否具备基本线上操作能力；
- 是否能通过命令定位常见故障；
- 是否知道命令风险，避免误删误改；
- 是否能组合管道处理日志。

## 常用命令

### 日志与文件

```bash
tail -f app.log                 # 实时查看日志
grep -n "ERROR" app.log         # 搜索错误
less app.log                    # 分页查看大文件
head -100 app.log               # 查看前 100 行
```

### 进程与端口

```bash
ps -ef | grep java              # 查看 Java 进程
top                             # 查看系统资源
top -Hp <pid>                   # 查看进程内线程 CPU
ss -lntp                        # 查看监听端口
```

### 磁盘与内存

```bash
df -h                           # 查看磁盘分区使用率
du -sh *                        # 查看目录大小
free -m                         # 查看内存
```

### 网络

```bash
ping <host>                     # 连通性测试
curl -v http://host:port/path    # HTTP 调试
```

## 标准回答

> 常用命令可以按场景回答：看日志用 `tail/grep/less`，看进程用 `ps/top`，查端口用 `ss`，看磁盘用 `df/du`，看内存用 `free`，调接口用 `curl`。实际排障时会结合时间窗口、关键字和服务链路分析，而不是只会单个命令。

## 易错点/总结

- 大文件不要直接 `vim` 或 `cat`；
- 删除、覆盖、重启类命令要确认环境和备份；
- `grep` 日志时注意时间范围；
- 线上命令输出要保留证据，方便复盘。

# 数据库三大范式是什么？

## 核心概念

数据库三大范式是什么？
第一范式：数据库表每一个列都是不可分割的原子数据项；
例如：单个列中同时包含手机号和邮箱，这个违反第一范式；

第二范式：在第一范式基础上，所有非主键列都需要与主键列相关，而不能跟部分主键相关；
例如：订单编号、产品编号、产品名称，其中订单编号和产品编号是联合主键，但是产品名称只依赖于产品编号，所以违反了第二范式；

第三范式：在第二范式基础上，所有非主键列都需要与主键列直接相关，不能传递依赖于主键。
例如：学号、系号、系名，其中学号是主键，系名依赖于系号，系号依赖于学号，但是这个系名不直接依赖学号，违反了第三范式。

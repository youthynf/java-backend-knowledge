# NOSQL与SQL有什么区别？

NOSQL与SQL有什么区别？
主要区别：
SQL数据库：关系型数据库存储结构化数据，这些数据逻辑上以行列二维表的形式存在，每一列代表数据的一种数据，每一行代表一个数据实体；
NOSQL数据库：非关系型数据库提供了不同于二维表的存储方式，其存储方式可以是JSON文档、哈希表或者其他方式；

实际应用：
SQL数据库：指关系型数据库，主要代表：SQL Server、Oracle、MySQL、PostgreSQL；
NOSQL数据库：指非关系型数据库，主要代表：MongoDB、Redis；

场景选择：
SQL数据库：支持ACID即原子性，一致性，隔离性和持续性；
NoSQL数据库：采用的是更宽松的模型BASE，即基本可用，软状态和最终一致性。

扩展对比：
SQL数据库：数据之间存在关联性，水平扩展较难，需要解决跨服务器JOIN、分布式事务等问题；
NOSQL数据库：数据之间无关联关系，这样容易扩展，比如Redis自带主从复制模式、哨兵模式、切片集群模式；

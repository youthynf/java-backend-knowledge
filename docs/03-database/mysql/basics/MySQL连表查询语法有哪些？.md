# MySQL连表查询语法有哪些？

## 核心概念

MySQL连表查询语法有哪些？
数据库主要有以下四种连表查询类型：
1.内连接（INNER JOIN）：
查询两个表中存在匹配关系的行；

2.左外连接（LEFT JOIN）：
返回左表中的所有行，即使右表中没有匹配的行，未匹配的右表列会包含NULL；

3.右外连接（RIGHT JOIN）：
返回右表中的所有行，即使坐标没有匹配的行，未匹配的左表列会包含NULL；

4.全外连接（FULL JOIN）：
在MySQL中，FULL JOIN 需要使用 UNION 来实现

# Redis 基础

本目录覆盖 Redis 入门概念：与 Memcache 的差异、远程连接安全基线、NoSQL 与 SQL 的对比。这些是面试 Redis 时的"暖场题"，回答质量直接影响面试官对深度的预期。

## 目录

- [Redis 与 Memcache 区别是什么](Redis与Memcache区别是什么？.md) — 同为内存缓存但设计哲学不同，Memcache 是纯缓存，Redis 是数据结构服务器
- [Redis 如何远程连接](Redis如何远程连接？.md) — 命令简单但生产重点是 bind、protected-mode、ACL、网络隔离
- [NoSQL 和 SQL 区别是什么](NoSQL和SQL区别是什么？.md) — 关系型与非关系型的哲学差异，CAP 与 ACID/BASE 取舍
